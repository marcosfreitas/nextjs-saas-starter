# WhatsApp Website

A warm, inviting, and modern theme inspired by WhatsApp.com's marketing website with its signature soft yellow backgrounds and clean design.

## Color Palette

- **Soft Yellow**: `#FCF5EB` - Primary background color, warm and inviting
- **WhatsApp Green**: `#25D366` - Primary brand color for CTAs and accents
- **Pure White**: `#FFFFFF` - Secondary background and card surfaces
- **Black**: `#000000` - Button borders, emphasis elements, and primary text
- **Dark Gray**: `#1F1F1F` - Primary text color
- **Medium Gray**: `#65676B` - Secondary text and subtle elements

## Typography

- **Headers**: Inter Bold / System UI Bold
- **Body Text**: Inter Regular / System UI Regular

## Visual Style

- Warm and inviting with soft yellow backgrounds
- Clean, minimal, and trust-building
- Generous white space
- Rounded buttons with thin black borders (1px)
- Button text is black and regular weight (not bold)
- Buttons include black icons on the right (typically arrow →)
- Friendly and approachable tone
- Marketing-focused aesthetic

## Best Used For

Marketing websites, landing pages, product showcases, consumer-facing web applications, promotional pages, feature announcements, and brand-focused experiences.

## Design Notes

This theme captures WhatsApp.com's welcoming aesthetic through its distinctive soft yellow (#FCF5EB) backgrounds paired with the iconic WhatsApp green. The warm cream background creates an inviting, approachable feel while maintaining professional clarity. Buttons feature bold black borders for clear visual distinction. Use the soft yellow generously for main backgrounds and the WhatsApp Green strategically for CTAs.
